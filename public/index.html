This website is a final project developed as one of the requirements for completing a Diploma 3 degree. This website functions as a machine reporting system for the Anodizing Department and was developed using the Laravel framework. The system aims to improve the quality of machine maintenance through features such as damage reporting, repair action logging, and real-time monitoring of report status. Additionally, there is a spare parts notification feature to ensure the availability of spare parts is maintained and to support the smooth running of the machine maintenance process.
